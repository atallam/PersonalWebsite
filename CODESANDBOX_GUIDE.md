# CodeSandbox Upload Guide

## Quick Steps to Run Your Portfolio on CodeSandbox

### Method 1: Direct Upload (Easiest)
1. Go to [codesandbox.io](https://codesandbox.io)
2. Click "Import Project" 
3. Drag your entire `personal-website` folder
4. Wait for automatic dependency installation
5. Your portfolio will start running!

### Method 2: Create New Sandbox
1. Go to [codesandbox.io](https://codesandbox.io)
2. Click "Create Sandbox" 
3. Choose "React TypeScript" template
4. Replace the template files with your project files

### Files to Upload
Your project is ready with all necessary files:
- `package.json` - Dependencies and scripts
- `src/` - All React components
- `index.html` - Entry point
- `vite.config.ts` - Build configuration
- `tailwind.config.js` - Styling configuration
- `tsconfig.json` - TypeScript configuration

### What CodeSandbox Will Do Automatically
- Install all dependencies from package.json
- Configure Vite development server
- Set up TypeScript compilation
- Process TailwindCSS
- Start your portfolio on port 3000

### Expected Results
- Beautiful gradient background
- Smooth animations and transitions
- Responsive design on all devices
- All sections: Hero, LinkedIn, Substack, GitHub
- Interactive navigation and scroll effects

### Troubleshooting
If anything doesn't work:
1. Check that all files were uploaded
2. Wait for dependency installation to complete
3. Refresh the browser tab
4. Check the CodeSandbox console for errors

### Share Your Portfolio
Once running, you can:
- Share the CodeSandbox link
- Deploy to Netlify directly from CodeSandbox
- Export to GitHub for deployment elsewhere
